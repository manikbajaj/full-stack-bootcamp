Explanation: Elements with float: left will align to the left side of the container. The next floated element will float next to it, as long as there’s enough space. These elements will sit inline until they fill the container width.

Explanation: When using floated elements, the parent container may collapse if it's not cleared. By applying a clearfix (as used here), the container expands to wrap around its floated children. This keeps the layout intact.

Float Right Example

Float Right 1
Float Right 2

Explanation: Elements with float: right will align to the right side of the container. Just like float left, multiple floated elements will line up next to each other but start from the right.

No Float Example

No Float

Explanation: Elements with float: none (default value) stay in the normal document flow. They don’t float and maintain the regular layout positioning, stacking as block-level elements.
