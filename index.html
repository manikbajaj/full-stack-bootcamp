CSS Inheritance Lecture

Basic Inheritance

Inherited from the parent.

Color is overridden, but other styles are inherited.

Non-Inherited Properties

Width and padding are not inherited.

Child box has its own width and padding.

Advanced Inheritance: Overriding Parent Styles

Font size, color, and line-height are inherited.

Font size is overridden, color and line-height are inherited.

Multiple Levels of Inheritance

Outer box styles inherited.

Middle box overrides some styles.

Inner box inherits styles from both parents.
