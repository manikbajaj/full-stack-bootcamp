CSS Rounded Corners Tutorial

Uniform Rounded Corners

border-radius: 20px;

In this example, the border-radius property is applied uniformly to all four corners, making them rounded by 20px.

Specific Corner Rounding

Top-left: 30px
Top-right: 30px
Bottom-left: 30px
Bottom-right: 30px

Here, different corners of each box have been rounded individually using border-top-left-radius, border-top-right-radius, border-bottom-left-radius, and border-bottom-right-radius properties.

Different Rounding for Each Corner

20px 40px 60px 80px

In this example, each corner has a different radius using border-radius with four values: top-left, top-right, bottom-right, and bottom-left.

Elliptical Rounding

50px / 20px

The border-radius property is applied using two values to create an elliptical shape. The first value is the horizontal radius, and the second is the vertical radius.

Circle (Fully Rounded)

By setting the border-radius to 50%, you can create a perfect circle if the width and height of the element are equal.
